xzModal ​

const xzModal: object

通用 UI 操作工具（基于 Element Plus）

封装常用用户交互方法，包括消息提示、弹窗确认、通知、加载遮罩等， 用于解耦业务逻辑与 UI 框架，提升可维护性与可测试性。
